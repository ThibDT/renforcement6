Partie 1: Interprétation de pseudo-code

Le signe ← signifie "prend la valeur".

Exercice 1

Variables : a, b (Integers)
c (String)

Début

a ← 0
b ← 5
c ← ''
Tant que a < b ajouter (b-a) à c
Si a modulo 2 vaut 0 incrémenter b avec un pas de 1
incrémenter a avec un pas de 1

Fin

a
b
c

Exercice 2

Variables : a (Integers)
b (Array)

Début

a ← 8
b ← [0, 1]
Pour i de 1 à a (inclus) ajouter à b la valeur de b[i-1] + b[i]

Fin

b

Exercice 3

Implémentez un algorithme mettant en application le chiffre de Vigenère.

Le chiffrement se fait en reportant l'indice de chaque lettre du message à ma clé (en reprenant la clé au début lorsque l'on atteint sa limite).
La lettre obtenue de cette façon définie la lettre de départ de mon alphabet.
Par exemple si mon message est "plop" et ma clé "codi" :
Je prend la lettre d'indice 0 dans "plop" (p) qui est la 16ème lettre de l'alphabet.
Je prend la lettre d'indice 0 dans ma clé "codi" (c).
Pour chiffrer cette lettre, mon alphabet correspondant commencera donc par c (cdefghijklmnopqrstuvwxyzab).
La 16ème lettre de cet alphabet est donc r.
On prend ensuite la lettre d'indice 1 dans mon message (l) qui est la 12ème lettre de l'alphabet.
Je prend la lettre d'indice 1 dans ma clé "codi" (o).
Mon alphabet correspondant commencera donc par o (opqrstuvwxyzabcdefghijklmn).
La 12ème lettre de cet alphabet est donc z.
On continue ainsi jusqu'à la fin du message.
